## Возможности

### Чат

- **Streaming** — ответ появляется в реальном времени, буква за буквой
- **DeepThink** — включает нативное reasoning:
  - DeepSeek: авто-переключение на `deepseek-reasoner`, поток `reasoning_content`
  - Claude: extended thinking через `thinking` parameter
  - Остальные: промпт-эмуляция
  - Ход мыслей показывается в отдельном блоке над ответом
- **Search** — перед ответом ищет в DuckDuckGo (топ-5 результатов), AI видит актуальную информацию из интернета
- **Контекст файлов** — открытый файл и структура проекта автоматически добавляются в контекст
- **Auto-run кода** — Python/JS код из ответа запускается, при ошибке AI исправляет (до 3 попыток)
- **История чатов** — шифруется AES-256-GCM, ключ привязан к машине

### Редактор и файлы

- Дерево проекта с иконками по типам файлов
- Создание, удаление, переименование файлов и папок
- Вкладки для нескольких файлов
- Код из чата можно отправить в редактор одной кнопкой
- `Ctrl+S` для сохранения, `Tab` для отступов

### Интерфейс

- Тёмная тема, frameless окно
- Кастомный titlebar в стиле macOS
- Статус-бар с индикатором состояния
- Поиск по файлам проекта и истории чатов (`Ctrl+F`)

## Установка

### Из исходников

```bash
git clone https://github.com/Cakein228/CakeAI.git
cd CakeAI
npm install
npm start
```

### Сборка дистрибутива

```bash
# NSIS-установщик + portable .exe
npm run build

# Только portable
npm run build-portable
```

Результат — в папке `dist/`.

### Требования

- [Node.js](https://nodejs.org/) 18+
- Windows 10/11 (x64)
- API-ключ любого поддерживаемого провайдера

## Использование

### Быстрый старт

1. Открой настройки — кнопка **⚙** в правом верхнем углу
2. Вставь API-ключ
3. Провайдер определится автоматически
4. Нажми **Применить**
5. Задай вопрос в строке ввода

### DeepThink

Кнопка **DeepThink** в строке ввода. При включении:

| Провайдер | Что происходит |
|-----------|---------------|
| DeepSeek | Переключается на `deepseek-reasoner`, reasoning стримится отдельно |
| Claude | Включается extended thinking (`budget_tokens: 10000`) |
| OpenAI, Gemini, Groq | Промпт-эмуляция: AI просит объяснять ход мыслей |

Блок **💭 Размышления** появляется над основным ответом.

### Веб-поиск

Кнопка **Search** в строке ввода. При отправке вопроса:

1. CakeAI ищет запрос в DuckDuckGo
2. Топ-5 результатов (заголовок + URL + сниппет) добавляются в контекст
3. AI отвечает с учётом найденной информации

### Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| `Enter` | Отправить сообщение |
| `Shift+Enter` | Перенос строки |
| `Ctrl+F` | Поиск |
| `Ctrl+S` | Сохранить файл |
| `Esc` | Закрыть окно |

### Пасхалка

Введи `0xteam` в строку чата.

## Архитектура

```
CakeAI/
├── main.js              # Main process: AI API, file system, crypto, web search
├── preload.js           # IPC bridge (contextIsolation: true)
├── package.json
├── README.md
└── src/
    ├── index.html       # UI
    ├── style.css        # Тёмная тема
    ├── reasoning.css    # Стили блока размышлений
    ├── chat.js          # Чат: streaming, reasoning, отправка
    ├── files.js         # Файловый менеджер
    ├── editor.js        # Редактор кода
    ├── settings.js      # Настройки, выбор модели
    ├── history.js       # Зашифрованная история
    ├── search.js        # Поиск по файлам/истории/веб
    ├── workspace.js     # Вкладки
    └── app.js           # Инициализация
```

### Безопасность

- `contextIsolation: true`, `nodeIntegration: false`
- API-ключи в `localStorage` (память Electron, не файл)
- История шифруется AES-256-GCM, ключ генерируется и хранится в `userData`
- Вывод AI экранируется от XSS перед рендерингом

### Поток данных

```
Пользователь вводит вопрос
        │
        ├──► [Search включён?] ──► DuckDuckGo API ──► результаты в контекст
        │
        ├──► [DeepThink включён?] ──► DeepSeek-reasoner / Claude thinking
        │
        ├──► Сборка system prompt + контекст файлов
        │
        ├──► Streaming запрос к API провайдера
        │         │
        │         ├──► stream-reasoning ──► блок "💭 Размышления"
        │         │
        │         └──► stream-chunk ──► блок ответа
        │
        └──► [Код в ответе?] ──► auto-run ──► исправление ошибок
```
